\subsubsection{Différences}
	\begin{itemize}
		\item Réalisation de l'inscription automatique aux sessions lors de l'enregistrement de l'utilisateur. 
		\item Utilisation de type de sessions afin de faciliter et de multiplier le choix des abonnements possibles. 
		\item Possibilité pour un utilisateur de s'inscrire à plus de un type de session par semaine. 
		\item Sélection du type de session parmi une liste. 
	\end{itemize}
